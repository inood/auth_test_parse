 3. Скрипт сбора событий этого и следующего месяца с сайта python.org
    >Разработайте скрипт на Python, который будет выводить в консоль
информацию о событиях этого и следующего месяца, анонсированных на
главной странице python.org. Вывод информации оформите по своему
усмотрению. Выбор библиотек на ваше усмотрение. Желательно завернуть
скрипт в докер контейнер, чтобы его можно было проверить на любой машине.
 
    - Скрипт парсит страницу https://www.python.org/events/python-events/
        
        и передает данные в шаблон Flask
    - Запустить можно из склонированного репозитория ```python parser.py```, 
    затем пройти по ссылке http://localhost:5000
    
    - Скрипт запакован в контейнер и доступен  в docker-hub: jetdj/parser:v0.1
    
      С установленным docker на компьютере можно запустить из консоли ```  docker run -d -p 5000:5000 jetdj/parser:v0.1 ```
